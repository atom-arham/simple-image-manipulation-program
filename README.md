# Simple Image Manipulation Program (SIMP)

A desktop application built with Python that lets you apply filters, blurs, and colour adjustments to your photos in real-time.

## Features
**Functionalities**
* Import
* Save
* Appearance (Darkmode and Lightmode)
  
**Filters**
* Invert, Grayscale, Sepia
* Sobel (Edge Detection)
* Cartoon, Emboss, Oil Paint

**Blurs**
* Simple, Gaussian, & Box Blur
* Median & Bilateral Blur
* Motion Blur

**Adjustments**
* **Light:** Brightness, Contrast, Blacks, Whites, Vignette
* **Colour:** Hue, Saturation, Temperature
* **Effects:** Noise (Normal, Uniform, Salt & Pepper)

## Built With

* **Python** – Core logic.
* **PySide6** – User Interface (GUI).
* **OpenCV** – Core Image manipulation and processing.
* **NumPy** – Matrix multiplication for fast filter calculations.
